# frontend

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve  # http://localhost:8080/
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
